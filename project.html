Bus Reservation System

C console application with login and sign up option.


This program has users' login info saved in a local txt file and allow users to sign up a new account if there isn't any matching login info. Users can check bus info, buy tickets, search for their bookings and cancel them. This is a simple console application and deals with all the data locally.


View this project on Github: https://github.com/nanntheintkham/Bus-Reservation-System
Product Order Processing

Java Console application with file processor


This program lets the user to load the csv file and reads the data which registers the order and product storage. The storage of products has separate containers and a generic class for all. The user can register the order, view the contents of containers in order, view the customers in order by the name and change the container size.


View this project on Github: https://github.com/nanntheintkham/Product_Storage
Personal Assistant

Java console application with JDBC.


This program has two features: health care and finance management. User data are stored in SQL database and keep track of them. So, data entered during the run time won't get lost after the application is closed.

For health care part, it will take gender, age, weight and height when the user first enters. They can check their bmi and calorie intakes for the day and check the record of them.

For finance part, they can calculate tax they have to pay according to their income and set the expense limit and it will warn them if they go over the limit.


View this project on Github: https://github.com/nanntheintkham/Personal-Assistant
